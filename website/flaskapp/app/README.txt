How to connect to gitpod docker container:
Link: https://gitpod.io/start/#seleniumhq-dockerseleni-onszkh4fhdg

Before running the docker container, first get the docker image id by typing the command below
docker images

This will show you a list of images stored in the selenium-docker, but they are not running
To run the docker, simply type the image id
